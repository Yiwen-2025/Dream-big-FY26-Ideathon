# Copilot Skills Library

This folder contains reusable Copilot skills for building demos,
web pages, mini games, and interactive presentations.

## 01 – Web Page Builder
Create simple single-page websites (HTML/CSS/JS), GitHub Pages ready.

## 02 – Mini Game Builder
Build browser-based mini games (Snake, Flappy Bird, 2048).

## 03 – Interactive Slide Deck
Create PPT-style interactive web presentations for demos or ideathons.

## 04 – Copy & Content Editor
Rewrite and polish text for pitches, emails, slides, and docs.

## 05 – QA & Delivery Checklist
Check readiness before sharing demos or submitting to judges.
